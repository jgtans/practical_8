# Notes Manager - Мобильное приложение для управления заметками

## 📱 Описание

**Notes Manager** - это кроссплатформенное мобильное приложение для создания, редактирования, удаления и поиска заметок. Приложение разработано для удобства ведения записей во время совещаний, собраний, конференций и других деловых встреч.

## ✨ Основные возможности

- ✅ **Создание заметок** - быстрое добавление новых записей с заголовком, датой и временем
- ✏️ **Редактирование** - изменение существующих заметок
- 🗑️ **Удаление** - удаление ненужных записей с подтверждением
- 🔍 **Поиск** - поиск заметок по ключевым словам в заголовке и содержимом
- 💾 **Два режима хранения**:
  - Локальная база данных SQLite
  - Файловая система (JSON файлы)
